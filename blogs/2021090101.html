JPA将mysql中的json转Map

mysql中可以使用text类型或者json类型（mysql5.7+）来存储json数据，使用JPA操作json类型数据时可以使用convert自动将json类型数据转换为对象。这里以Map类型为例，记录在JAP中如何使用convert转换json类型数据。

1. 设置字段类型为Map<String, String>，并添加注解@Convert

@Convert注解用来指定类型转换器，转换器需要实现AttributeConverter接口

@Convert(converter = JpaMapConverter.class)
@Column(name = “map_fields”, columnDefinition = "json")
private Map<String, String> mapFileds;

2. 转换器类实现

转换器类需要实现AttributeConverter接口，并重写两个方法。这里对象和json的相互转换使用了fastjson。

public static class JpaMapConverter implements AttributeConverter<Map<String, String>, String> {
	@Override
	public String convertToDatabaseColumn(Map<String, String> map) {
		if(map == null) {
			return "{}";
		}
		return JSON.toJSONString(map);
	}

	@Override
	public Map<String, String> convertToEntityAttribute(String value) {
		if(value == null || value.isEmpty()) {
			return Collections.emptyMap();
		}
		return JSON.parseObject(value, HashMap.class);
	}
}
