centos7安装mycli

mycli是一个开源的增强mysql客户端命令行工具，本文介绍如何安装及使用它。

安装

mycli是使用python开发出来的，需要python3环境。

yum install python3 python3-devel python3-pip
pip3 install mycli
# 或者使用国内源
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple mycli

配置

mycli的用户配置文件是~/.myclirc，使用别名访问需要在文件最下方的[alias_dsn]段增加配置。

[alias_dsn]
# example_dsn = mysql://[user[:password]@][host][:port][/dbname]
# 文件最下方增加这一行实现别名访问
local = mysql://root:root@localhost:3306
