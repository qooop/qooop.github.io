centos7安装mycli

安装

mycli是使用python开发出来的，需要python3环境。

yum install python3 python3-devel python3-pip
pip3 install --upgrade pip
pip3 install mycli

配置

mycli的用户配置文件是~/.myclirc。

[alias_dsn]
# example_dsn = mysql://[user[:password]@][host][:port][/dbname]
# 文件最下方增加这一行实现快速访问
local = mysql://root:root@localhost:3306
