Ract-堆的数据结构

最近看了下react源码，其中scheduler项目中有一个ShcedulerMinHeap.js的文件，描述了一个堆的数据结构。我觉得挺有收获的，记录一下。

堆是一个数组构成的二叉树，具有以下特性：

  • 堆中某个结点的值总是不大于或不小于其父结点的值；
  • 堆总是一棵完全二叉树。

堆中的每一个元素可以是任意类型的对象，往堆里边插入一个元素会经过以下计算：

1. 往数组中push

export function push(heap: Heap, node: Node): void {
  const index = heap.length; //插入的元素的下标
  heap.push(node);
  siftUp(heap, node, index);
}

2. 与父元素比较大小判断是否需要交换位置（排序）

function siftUp(heap, node, i) {
  let index = i;
  while (index > 0) {
    const parentIndex = (index - 1) >>> 1; //右移运算，相当于除2取整
    const parent = heap[parentIndex];
    if (compare(parent, node) > 0) { //判断父元素和当前元素大小，如果父元素大，则交换位置
      // The parent is larger. Swap positions.
      heap[parentIndex] = node;
      heap[index] = parent;
      index = parentIndex;
    } else {
      // The parent is smaller. Exit.
      return;
    }
  }
}
