fish命令的使用

简介

fish是friendly interactive shell的简写，友好交互shell的意思。

安装

以下为常见系统的安装，其他系统安装方式参考官方文档 (opens new window)

# debian
apt install fish

# centos
yum install fish

# termux
pkg install fish

# homebrew
brew install fish
